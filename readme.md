# Computer graphics - Basic course

## Multithreaded Raytracer written in Java

features:
* implemented multiple shapes such as:
    - Spheres
    - Cylinders
    - Disks
    - Cones
* multicore optimized rendering
* multiple materials such as:
    - diffuse
    - glass
    - glossy/reflective
    - textures (todo)
* skyboxes (todo)
* bounding boxes (todo)
